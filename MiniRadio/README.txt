MiniRadio - простая программа с открытым исходным кодом.
Основные возможности:
1) 75 радиостанций(в первой версии. В дальнейшом список станций будет обновляться.)
2) Поиск клипов в youtube, музыки в Вконтакте, артиста в LastFm(при условии, что радиостанция
 отдает теги).
Список возможностей будет поплняться.

https://miniradio.codeplex.com/
